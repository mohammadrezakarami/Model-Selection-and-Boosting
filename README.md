# Model Selection and Boosting

This repository contains notebook implementations related to model validation, hyperparameter search, and boosting methods.

## Repository Contents

- `k_fold_cross_validation.ipynb`
- `grid_search.ipynb`
- `xg_boost.ipynb`
- `Social_Network_Ads.csv`: dataset used in the experiments

## Topics Covered

- k-fold cross validation
- grid search
- model selection
- XGBoost
- evaluation of supervised learning workflows

## Getting Started

1. Clone the repository.
2. Open the notebooks in Jupyter.
3. Install common machine learning dependencies such as scikit-learn and XGBoost.
4. Run the notebooks to reproduce the selection and boosting experiments.

## Notes

This repository is intended as an educational set of experiments on validation and performance improvement techniques.
